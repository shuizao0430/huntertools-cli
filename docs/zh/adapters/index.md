# 招聘适配器

运行 `huntertools list` 查看当前真实可用的命令注册表。

## 当前公开产品面

| 站点 | 命令 |
|------|------|
| `linkedin` | `search` `timeline` `people-search` `profile` `recruiter-project-list` `recruiter-project-members` `recruiter-saved-searches` `message` `save-to-project` `tag` `notes` `batch-message` `inbox-list` `inbox-msg` `inbox-reply` `batch-reply` `stats` `follow-up-queue` `follow-up-batch-reply` `export-follow-up` |

## 说明

- HunterToolsCLI 已经从通用多站点工具收口为招聘工作流产品。
- 对外支持范围以 `huntertools list` 为准。
- 当前重点支持的是 LinkedIn 与 LinkedIn Recruiter。
